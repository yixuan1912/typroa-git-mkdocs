### file(显示文件行数)

​	**`file [路径]`** 可以显示文件的第一行

- 参数 `-n`可以指定显示的行数, 默认一行

### wc(显示行数,字数)

​	计算文件中 `行` , `字` 和 `字符数量`

- 参数有`l`  `w`  `c` 分别用于只显示**行数, 字数, 字符数**

### history (之前用的命令)

- 可以查看之前执行的命令的列表

### 编辑命令行(快捷编辑)

| 快捷方式        | 描述                               |
| --------------- | ---------------------------------- |
| Ctrl + a        | 跳到命令行的开头                   |
| Ctrl + e        | 跳到命令行的末尾                   |
| Ctrl + u        | 将光标处到命令行开头的内容清除     |
| Ctrl + k        | 将光标处到命令行末尾的内容清除     |
| Ctrl + 向左箭头 | 跳到命令行中前一个字的开头         |
| Ctrl + 向右箭头 | 跳到命令行中下一个字的开头         |
| Ctrl + r        | 在历史记录列表中搜索某一模式的命令 |
