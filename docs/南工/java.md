### 方法重载

方法重载指同一个类中的多个方法之间的关系，需满足下列条件

- 多个方法在同一类中
- 多个方法具有相同的方法名
- 多个方法的参数不相同，类型不同或者数量不同

特点：

- 重载仅对应方法的定义，与方法的调用无关，调用方式参照标准格式
- 重载仅针对用一个类中方法的名称与参数进行识别，与返回值无关，换句话说就是不能通过返回值来判断两个方法是否互相构成重载
